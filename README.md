# keymaps

My keyboards, keymaps, and utils for qmk_firmware

## Usage

    ./qmk KEYBOARD build
    ./qmk KEYBOARD flash

## CLI Development

`keymaps` and `keyboards` contain directory mappings to the qmk keyboard and keymap.

### Submodules

    git submodule update --init --recursive
